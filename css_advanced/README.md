# CSS Advanced

## Project Description

This project is part of the ALCHE Web Development curriculum.

The goal of this project is to take the HTML structure created in the HTML Advanced project and apply CSS styling to recreate a webpage based on a Figma design.

The project focuses on improving skills in:
- CSS styling
- Layout design
- Responsive design
- Web page visual implementation

## Objectives

By completing this project, I will learn how to:

- Style a webpage using CSS
- Transform an HTML structure into a visual design
- Work with fonts, colors, spacing, and positioning
- Understand how professional web pages are built from designer files

## Technologies Used

- HTML5
- CSS3
- Figma
