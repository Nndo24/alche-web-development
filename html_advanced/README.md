# HTML Advanced

## Project Description

This project is part of the **ALCHE Web Development** curriculum.

The goal is to recreate a webpage from a Figma design using only semantic HTML. At this stage, no CSS or JavaScript is used. The focus is on writing clean, accessible, and well-structured HTML.

## Objectives

- Learn semantic HTML
- Structure a webpage correctly
- Follow a Figma design
- Prepare the project for future CSS styling

## Technologies

- HTML5
- Git
- GitHub
